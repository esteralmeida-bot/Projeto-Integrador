FONTES E TIPOGRAFIAS

Projeto Integrador

Propriedades basicas
Font-Family: Define a familia da fonte;
Font-Size: é o tamanho do texto.
Color: é a cor.
Line-height: altura da linha, é o espaçamento.
Text-aling: é o posicionamento do texto. Text-transform: Vai alterar o texto para
maiúsculo (uppercase) ou minusculo (lowercase)
Letter-spacing: espaçamento horizontal entre as palavras.
Text-shadow: efeito de sombra no texto.
